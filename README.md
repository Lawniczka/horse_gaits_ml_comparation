# Wyjaśnialna analiza koncepcji chodu konia w modalności wizyjnej
*Explainable horse gait concept analysis in the visual modality*

## Kolejność uruchamiania

Aby poprawnie przygotować dane i uruchomić modele, należy zachować następującą kolejność:

1. **Feature extraction**  
    W pierwszym kroku należy uruchomić skrypty znajdujące się w folderze `feature_extraction` **według kolejności ich nazw**

2. **Modele**  
    Notebooki w folderze `models` zawierają kompletny pipeline: tworzenie, trenowanie i testowanie modeli, oraz zapisanie wyników w formie macierzy.

3. **Metryki**  
    Notebook `metrics` w folderze `models` definiuje metody wyliczania metryk z otrzymanych macierzy
